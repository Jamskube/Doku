## Discoveries
- **Arbitrage structurel IA locale ARM64** : « charger tes GGUF toi-même » (Ollama/llama.cpp) = **CPU-only** sur X Elite ; le **NPU** ne mange que de l'ONNX/QNN pré-converti (catalogue fermé, Foundry Local) → « UX Ollama » et « NPU » **mutuellement exclusifs**. Le NPU **n'accélère pas** (borné bande passante mémoire ~30 tok/s) ; son seul gain = **autonomie batterie**.
- **Ollama a un build ARM64 natif** → bundlable en **sidecar Tauri** (`externalBin` + capability `shell`), Rust minimal préservé, UX pull/run/switch gratuite. Bug détection NPU spécifique Surface Pro 11 (Foundry Local #797).
- **PDF sur ARM64** : lecture via **PDF.js** (JS/WASM, indépendant de l'archi, worker sous CSP `worker-src blob:`) ; export via **`window.print`/`PrintToPdfAsync`** (WebView2 ARM64, piège #5199) ; binaires **PDFium ARM64** existent (bblanchon) si besoin natif.

## Fixed
- **9.2 (revue `code-reviewer`)** : index de recherche **périmé** (une note créée/rechargée n'apparaissait pas) → `invalidateSearchDoc` capte création + reload (`applyDiskContent`) ; **builds d'index concurrents** sous frappe rapide → mémoïsation de la promesse en vol par dossier

## Discoveries
- **Piloter CM6 après ouverture d'onglet** (montage async) : poser une intention (`pendingReveal`) + la consommer dans un `$effect` déclaré **après** le switch d'onglet — un appel direct après `openPath` court-circuite le montage
- **Offsets recherche** : ceux d'un extrait fenêtré ≠ position document → porter la colonne brute (`col`) ; `toLowerCase` peut changer la longueur (U+0130)
- **Vérif Playwright** : le `localStorage` persisté survit entre sessions et inverse un raccourci toggle ; un bouton adjacent à CM peut faire échouer le clic → repli `browser_evaluate(() => el.click())`

## Decisions
- **Recherche = index en mémoire** (scan-once + cache + invalidation, ADR-0007), aucune lib tierce ; matching sous-chaîne casse-insensible (pas de regex)
- Le **cadre d'occurrence** (`.cm-search-flash`) plutôt qu'une simple sélection — répond à « qu'il soit encadré »
- **Sprint 9 = Epic 10 Export** (PDF + HTML autonome, DOCX stretch) ; 10.1 spike pipeline PDF en tête ; export PDF **non testable en dev navigateur** (WebView2 print) → vérif natif
