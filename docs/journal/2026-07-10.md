## Discoveries
- CM6 : un **`ViewPlugin` ne peut pas fournir de décoration `replace` block-level / traversant des lignes** → passer par un **`StateField`** (widget-bloc tableau)
- CM6 : un **block-replace doit être aligné sur des frontières de ligne** ; les nœuds bloc Lezer démarrent **après l'indentation** (mi-ligne) → ancrer explicitement
- CM6 : le **clic sur un widget** se capte via un listener `mousedown` **dans `toDOM(view)`** (motif `CheckboxWidget`), pas un `domEventHandlers` global (qui ne le voit pas fiablement)
- Suppression de fichiers : **scoper `fs:allow-remove`** au sous-arbre app (`$APPDATA/snapshots/**`) + garde runtime — jamais `**`
- Restauration : ne snapshotter le courant que si dirty (sinon doublons) — retour smoke
- Op : ne pas laisser un `npm run dev` navigateur tourner pendant un smoke natif (`strictPort` → module Vite périmé)
