## Discoveries
- Milkdown/remark-stringify réécrit **chaque** fichier au round-trip (tables re-paddées, `_it_`→`*it*`, échappements) — rédhibitoire pour éditer des fichiers existants
- CM6 : 500 Ko chargé en 27 ms vs 1 727 ms ; frappe 17 vs 69 ms/frame — le live preview est aussi le comportement exact spécifié par FR-3
- KUDE (repo de référence) : pièges documentés — I/O Rust custom à lignes numérotées, Google Fonts en @import, couche d'alias CSS legacy, thème par défaut incohérent
- CM6 ne rend que le viewport : les vérifications DOM (Playwright) doivent scroller la cible en vue ; un conteneur scrollable externe conserve son scrollTop entre `setState`

## Decisions
- Stack : Tauri 2 + Svelte 5 + TS (ADR-0001) · Moteur : CM6 live preview, mode source = même éditeur via Compartment (ADR-0002) · Snapshots : `%APPDATA%\Doku` centralisé (ADR-0003) · I/O : plugins officiels, zéro Rust custom (ADR-0004)
- UX : WYSIWYG à la Typora · sidebar repliée totalement invisible · mini-barre de sélection oui · compteur de mots non · thème crème par défaut
