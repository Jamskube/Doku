## Discoveries
- CodeMirror : `state.doc.toString()` renvoie toujours `\n` (le facet `lineSeparator` ne joue que sur le split) → sérialiser avec la fin de ligne détectée
- Tauri v2 : rappeler `window.close()` depuis `onCloseRequested` ne se propage pas (surtout en release) → `window.destroy()`
- Comportements natifs (fermeture fenêtre) diffèrent dev/release → smoke-tester en release
- DOMPurify incompatible happy-dom → jsdom
- CodeMirror : ne pas muter le DOM d'un `WidgetType` via `replaceWith` (clobbered) → conteneur stable
- Icône barre des tâches Windows = double cache (exe gravé au build + cache Windows)
- La CLI Tauri auto-ajoute les features requises à `Cargo.toml` (ex. `protocol-asset` via `assetProtocol`) — à committer
