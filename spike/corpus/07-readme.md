# Doku

Petite application pour **ouvrir, lire et éditer des fichiers Markdown**, avec pour ambition de supporter plus tard d'autres formats de documents (PDF, etc.).

## État du projet
🌱 Initialisation — la stack technique n'est pas encore choisie. Voir `docs/planning/` (à venir) pour le PRD et l'architecture.

## Structure
| Dossier | Rôle |
|---|---|
| `src/` | Code source |
| `docs/` | Documentation, journal de bord, archives |

## Contexte IA
Le fichier `AGENTS.md` contient le contexte pour les assistants de code — à lire en début de session.
